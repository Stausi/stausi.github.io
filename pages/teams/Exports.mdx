## Team Data Structure

- owner: `table`
    - source: `number`
    - identifier: `string`
    - name: `string`
- uuid: `string`
- name: `string`
- members: `table`
    - source: `number`
    - identifier: `string`
    - name: `string`
- invites: `table`
    - source: `number`
    - identifier: `string`
    - name: `string`

## Get Team Exports

<Tabs items={["uuid", "source", "identifier"]}>
    <Tab>
        Returns the team from the Team UUID

        ```lua
        local team = exports.st_steams:getTeamFromUUID(uuid)
        ```

        - uuid: `string`

    </Tab>

    <Tab>
        Returns the team from the source of a user

        ```lua
        local team = exports.st_steams:getTeamFromSource(source)
        ```

        - source: `number`

    </Tab>

    <Tab>
        Returns the team from the users identifier (steam)

        ```lua
        local team = exports.st_steams:getTeamFromIdentifier(identifier)
        ```

        - identifier: `string`

    </Tab>
</Tabs>

## LeaveTeam

Makes the player leaves the team

```lua
exports.st_steams:leaveTeam(source)
```

- source: `number`

## AddPlayerToTeam

Makes the player leaves the team

```lua
exports.st_steams:AddPlayerToTeam(source, target)
```

- source: `number` (The user which adds the player to the team)
- target: `number` (The user which should join the team)

## IsPlayerInTeam

Returns if the user is in a team or not

```lua
local isInTeam = exports.st_steams:IsPlayerInTeam(source)
```

- source: `number`

## createTeam

Create a team for the user with custom generated name.

```lua
local isInTeam = exports.st_steams:createTeam(source)
```

- source: `number`
