# Clothing Script

## OpenClothingMenu

Opens the clothing menu

```lua
exports.st_clothing:OpenClothingMenu(openType, hasWardrobe)
```

- openType: `string | nil` - Which type of shop it should open.
    - command
    - plastic
    - barber
    - clothes

- hasWardrobe: `boolean | nil` - If the opened clothing should have a wardrobe or not.
