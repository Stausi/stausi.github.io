## StartLockpickMinigame

Starts a lockpick minigame using UI

Example of using this with synced function, with a return value

```lua
local hasWon = exports.st_minigames:StartLockpickMinigame(corrects, timeMultiplier)
```

- corrects: `number` (How many corrects the player must have)
- timeMultiplier: `number` (How fast the arrow moves)

Example of using this with asynced callback function

```lua
exports.st_minigames:StartLockpickMinigame(corrects, timeMultiplier, function(hasWon)
end)
```

- corrects: `number` (How many corrects the player must have)
- timeMultiplier: `number` (How fast the arrow moves)


## Example
```lua
local corrects = 2           
local timeMultiplier = 2 
exports.st_minigames:StartLockpickMinigame(corrects, timeMultiplier, function(hasWon)
    if hasWon then
        print("Lockpick successful!")
    else
        print("Lockpick failed!")
    end
end)
```
