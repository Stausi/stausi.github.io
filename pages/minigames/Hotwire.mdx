## StartHotwire

Starts a hotwire minigame using UI

Example of using this with synced function, with a return value

```lua
local hasWon = exports.st_minigames:StartHotwire(time)
```

- time: `number` (Times in seconds to complete the minigame)

Example of using this with asynced callback function

```lua
exports.st_minigames:StartHotwire(time, function(hasWon)
end)
```

- time: `number` (Times in seconds to complete the minigame)

## Example
```lua
local time = 60
exports.st_minigames:StartHotwire(time, function(hasWon)
    if hasWon then
        print("Hotwire successful!")
    else
        print("Hotwire failed!")
    end
end)
```
