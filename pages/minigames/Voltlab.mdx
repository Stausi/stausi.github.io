## Voltlab

Starts a Voltlab minigame using UI and Scaleforms

Example of using this with synced function, with a return value

```lua
local hasWon = exports.st_minigames:Voltlab(time)
```

- time: `number` (Times in seconds to complete the minigame)

Example of using this with asynced callback function

```lua
exports.st_minigames:Voltlab(time, function(hasWon)
end)
```

- time: `number` (Times in seconds to complete the minigame)


## Example
```lua
local time = 60  
exports.st_minigames:Voltlab(time, function(hasWon)
    if hasWon then
        print("Voltlab minigame successful!")
    else
        print("Voltlab minigame failed!")
    end
end)
```
