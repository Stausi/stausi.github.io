## StartDrilling

Starts a drilling minigame using scaleforms

Example of using this with synced function, with a return value

```lua
local hasWon = exports.st_minigames:StartDrilling()
```

Example of using this with asynced callback function

```lua
exports.st_minigames:StartDrilling(function(hasWon)
end)
```

## Example
```lua
exports.st_minigames:StartDrilling(function(hasWon)
    if hasWon then
        print("Drilling successful!")
    else
        print("Drilling failed!")
    end
end)
```
