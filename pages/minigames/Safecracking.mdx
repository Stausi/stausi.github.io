## SafeCrack

Starts a safecrack minigame using Scaleforms

Example of using this with synced function, with a return value

```lua
local hasWon = exports.st_minigames:SafeCrack(status, locks)
```

- status: `boolean` (If safecrack UI should be shown or not)
- locks: `number` (How many locks the player should crach)

Example of using this with asynced callback function

```lua
exports.st_minigames:SafeCrack(status, locks, function(hasWon)
end)
```

- status: `boolean` (If safecrack UI should be shown or not)
- locks: `number` (How many locks the player should crach)
