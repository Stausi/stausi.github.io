## StartThermiteMinigame

Starts a thermite minigame using UI

Example of using this with synced function, with a return value

```lua
local hasWon = exports.st_minigames:StartThermiteMinigame(mode, time)
```

- mode: `number` (How many boxes which should be shown. Standard 5 which is 5x5=25 boxes)
- time: `number` (Times in seconds to complete the minigame)

Example of using this with asynced callback function

```lua
exports.st_minigames:StartHotwire(mode, time, function(hasWon)
end)
```

- mode: `number` (How many boxes which should be shown. Standard 5 which is 5x5=25 boxes)
- time: `number` (Times in seconds to complete the minigame)


## Example
```lua
local mode = 5      
local time = 60     
exports.st_minigames:StartThermiteMinigame(mode, time, function(hasWon)
    if hasWon then
        print("Thermite minigame successful!")
    else
        print("Thermite minigame failed!")
    end
end)
```
