## StartHackingMinigame

Starts a hacking minigame using UI

Example of using this with synced function, with a return value

```lua
local hasWon = exports.st_minigames:StartHackingMinigame(style, timeout, interval)
```

- style: `string` (Styles below)
    - numeric
    - alphabet
    - alphanumeric
    - greek
    - braille
    - runes
- timeout: `number` (Time in second when the minigame stops and make the user lose)
- interval: `number` (Interval for each time the UI moves)

Example of using this with asynced callback function

```lua
exports.st_minigames:StartHackingMinigame(style, timeout, interval, function(hasWon)
end)
```

- style: `string` (Styles below)
    - numeric
    - alphabet
    - alphanumeric
    - greek
    - braille
    - runes
- timeout: `number` (Time in second when the minigame stops and make the user lose)
- interval: `number` (Interval for each time the UI moves)
